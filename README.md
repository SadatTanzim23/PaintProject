#Paint Project
##One Piece themed 

A One Piece–inspired paint application built in Python that lets users freely draw, sketch, and express creativity with a fun pirate-themed interface.

What does the program do?

* Allows users to draw freely on a digital canvas using mouse input.
* Provides multiple drawing tools (e.g., brush, eraser, color selection).
* Includes a One Piece–themed interface for a fun and engaging user experience.
* Handles real-time rendering and user events using Pygame for smooth interaction.
